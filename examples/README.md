# Exemples (zone de dépôt)

Dépose ici tes documents de référence — ils servent de **canevas** au projet.

## `trames/`
Exemples de **trames d'interview** (`.docx`). Servent de modèles pré-remplis
pour la saisie (US1.3 — import de trame). Dépose un fichier représentatif :
on s'en sert pour caler le parser (Heading → thème, listes → questions).

## `templates/`
**Templates PPT** corporate (`.pptx`) + decks exemples de restitution.
Utilisés plus tard (incréments 4-5) : le template donne la charte, les
exemples donnent la trame narrative du deck généré.

> Ces dossiers ne sont pas versionnés s'ils contiennent des données client —
> garde ça en tête avant tout commit.
